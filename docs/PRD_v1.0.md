## 1. 项目概述

### 1.1 项目目标

打造 AI 驱动的全流程内容创作系统，实现一次选题多平台内容自动生成。

### 1.2 核心价值

每次创作节省约 3.5 小时。

### 1.3 目标用户

个人内容创作者，运营抖音/视频号/公众号/小红书等多个平台。

---

## 2. 功能需求

### 2.1 核心工作流

```
选择模板 → 输入主题 → AI 联网搜索 → 推荐选题 → 选择选题 → 生成内容 → 保存到历史
```

### 2.2 功能模块

| 模块 | 功能描述 | 优先级 |
|------|---------|--------|
| 模板选择 | 选择预设内容模板 | P0 |
| AI 搜索 | 微博热搜 + 知乎热榜 | P0 |
| 选题推荐 | 3-5 个选题角度 | P0 |
| 内容生成 | 抖音 + 公众号 + 小红书 | P0 |
| 配图生成 | 最多 3 张 AI 配图 | P0 |
| 历史管理 | 查看/复制/下载 | P0 |

---

## 3. 非功能需求

### 3.1 性能要求

- 搜索响应时间：< 30 秒
- 内容生成时间：< 60 秒
- 配图生成时间：< 10 秒/张

### 3.2 质量要求

- 内容可用性：> 80%
- 系统稳定性：> 95%

---

## 4. 验收标准

### 4.1 MVP 验收

- [ ] 能正常搜索微博/知乎热榜
- [ ] 能生成 3-5 个选题
- [ ] 能生成三平台内容
- [ ] 能生成配图（最多 3 张）
- [ ] 能保存和查看历史
