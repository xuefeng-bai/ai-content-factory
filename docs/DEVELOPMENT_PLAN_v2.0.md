### Phase 2：Prompt 配置化 + 内容生成（🔄 进行中）

**时间：** 2026-03-14 ~ 2026-03-20（7 天）

**核心功能：**
- 🔄 Prompt 数据库存储（prompts, prompt_versions, prompt_test_logs）
- 🔄 Prompt 管理界面（查看、编辑、测试、版本历史）
- 🔄 AIService 重构（从数据库加载 Prompt）
- ⏳ 选题推荐（AI 推荐 3-5 个选题）
- ⏳ 内容生成（抖音 + 公众号 + 小红书）
- ⏳ 配图生成（公众号 16:9 + 小红书 3:4）
- ⏳ 历史记录（保存和查看）

**交付物：**
- `backend/app/models/prompt.py`
- `backend/app/models/prompt_version.py`
- `backend/app/api/prompts.py`
- `backend/app/ai/service.py`
- `backend/app/api/topics.py`
- `backend/app/api/content.py`
- `backend/app/api/images.py`
- `backend/app/api/history.py`
- `frontend/src/pages/Prompts.jsx`
- `frontend/src/pages/PromptEdit.jsx`
- `frontend/src/pages/PromptTest.jsx`
- `frontend/src/pages/Preview.jsx`
- `frontend/src/pages/History.jsx`

**完成状态：** 🔄 0%
