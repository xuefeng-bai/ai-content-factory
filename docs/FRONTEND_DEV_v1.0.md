## 2. 页面设计

### 2.1 页面清单

| 页面 | 路径 | 功能 |
|------|------|------|
| 首页 | / | 模板选择 + 主题输入 |
| 搜索结果 | /search | 热点趋势 + 选题推荐 |
| 内容预览 | /preview | 三平台内容 + 配图 |
| 历史列表 | /history | 查看/复制/下载 |

### 2.2 核心组件

```
src/
├── components/
│   ├── TemplateSelector.jsx    # 模板选择器
│   ├── SearchResult.jsx        # 搜索结果
│   ├── ContentPreview.jsx      # 内容预览
│   ├── HistoryList.jsx         # 历史列表
│   └── ImageGallery.jsx        # 图片展示
├── stores/
│   └── useStore.js             # 全局状态
└── services/
    └── api.js                  # API 调用
```

## 3. 交互设计

### 3.1 模板选择

- 卡片式展示
- 鼠标悬停显示详情
- 自定义配置弹窗

### 3.2 内容预览

- 标签页切换（抖音 | 公众号 | 小红书）
- 公众号文章支持折叠/展开
- 配图用缩略图，点击放大

### 3.3 一键复制

- 抖音：复制文字
- 公众号：复制 Markdown / 下载文件
- 小红书：文案复制 + 图片下载
