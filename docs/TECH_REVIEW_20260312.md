## 🙋 技术实现逐项讨论

### 模块 1：AI 联网搜索

#### 技术方案

**微博热搜：**
- 方案：爬虫（https://s.weibo.com/top/summary）
- 难度：🟡 中
- 风险：反爬机制（限流）
- 对策：设置请求间隔 + User-Agent 轮换

**知乎热榜：**
- 方案：爬虫（https://www.zhihu.com/hot）
- 难度：🟡 中
- 风险：反爬机制（登录验证）
- 对策：模拟登录 + Cookie 管理

**备选方案：**
- 用 web_search 工具搜索"微博热搜"和"知乎热榜"
- 难度：🟢 低
- 风险：数据可能不够实时

#### 技术门槛

| 问题 | 难度 | 解决方案 | 状态 |
|------|------|---------|------|
| 微博反爬 | 🟡 中 | 请求间隔 + UA 轮换 | ✅ 可解决 |
| 知乎登录验证 | 🟡 中 | 模拟登录 + Cookie | ✅ 可解决 |
| 数据实时性 | 🟢 低 | 定时刷新缓存 | ✅ 可解决 |

### 模块 2：选题推荐算法

#### 技术方案

**输入：** 搜索结果（JSON 格式）
**处理：** DashScope Qwen API
**输出：** 3-5 个选题（固定格式）

**Prompt 设计：**
```
你是专业的内容策划专家。请根据以下搜索素材，推荐 3-5 个选题角度。

搜索素材：
{search_results}

请按以下格式输出：
1. 选题标题
   类型：[工具推荐/认知差/案例分享/观点输出]
   优势：[一句话说明]
   推荐度：⭐⭐⭐⭐⭐

要求：
- 选题要有吸引力
- 类型要多样化
- 优势要具体
```

#### 技术门槛

| 问题 | 难度 | 解决方案 | 状态 |
|------|------|---------|------|
| 选题质量不稳定 | 🟡 中 | 优化 Prompt + 示例 | ✅ 可解决 |
| 输出格式不固定 | 🟡 中 | 结构化 Prompt + 后处理 | ✅ 可解决 |
| 推荐度评分主观 | 🟢 低 | AI 自动评分 | ✅ 可解决 |

### 模块 3：平台智能体

#### 技术方案

**抖音智能体：**
- Prompt：口语化 + 钩子开头 + 金句结尾
- 字数控制：Prompt 中明确（如"500-600 字"）
- 时长估算：字数 / 200 字/分钟

**公众号智能体：**
- Prompt：结构化 + Markdown 格式
- 字数控制：Prompt 中明确
- 配图位置：AI 自动标注 `![配图](url)`

**小红书智能体：**
- Prompt：种草风格 + emoji + 标签
- 标题：吸引眼球（AI 自动生成 3 个备选）
- 封面：AI 文生图

#### 技术门槛

| 问题 | 难度 | 解决方案 | 状态 |
|------|------|---------|------|
| 字数控制不精确 | 🟡 中 | Prompt + 后处理校验 | ✅ 可解决 |
| 口语化程度难把握 | 🟡 中 | 示例 + 微调 | ✅ 可解决 |
| Markdown 格式错误 | 🟢 低 | 格式校验 | ✅ 可解决 |

### 模块 4：AI 配图生成

#### 技术方案

**技术栈：** DashScope 文生图 API
**数量限制：** 每次最多 3 张
**比例：**
- 公众号：16:9（横版）
- 小红书：3:4（竖版封面）

**Prompt 设计：**
```
公众号配图提示词：
"科技感，商务风格，AI 人工智能主题，蓝色调，简洁大气"

小红书封面提示词：
"小红书爆款风格，大字报，醒目文字，橙色调，吸引眼球"
```

#### 技术门槛

| 问题 | 难度 | 解决方案 | 状态 |
|------|------|---------|------|
| 图片质量不稳定 | 🟡 中 | 优化提示词 + 多次生成 | ✅ 可解决 |
| 生成时间较长 | 🟡 中 | 异步处理 + 进度提示 | ✅ 可解决 |
| 图片与内容不匹配 | 🟡 中 | 从内容提取关键词 | ✅ 可解决 |
| 成本问题 | 🟢 低 | 限制最多 3 张 | ✅ 已解决 |

### 模块 5：内容生成与预览

#### 技术方案

**前端：** React + Ant Design
**展示方式：** 标签页切换（抖音 | 公众号 | 小红书）
**复制功能：**

- 代码高亮：Prism.js（Markdown 预览）
- 图片预览：Ant Design Image
- 一键复制：navigator.clipboard API

#### 技术门槛

| 问题 | 难度 | 解决方案 | 状态 |
|------|------|---------|------|
| Markdown 渲染复杂 | 🟡 中 | react-markdown 库 | ✅ 可解决 |
| 图片下载跨域 | 🟡 中 | 后端代理下载 | ✅ 可解决 |
| 一键复制兼容性 | 🟢 低 | 降级方案（手动选择） | ✅ 可解决 |
